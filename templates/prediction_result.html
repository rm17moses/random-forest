Pollution Index

The pollution index used to measure contamination for this model is the Modified contamination factor (mCdeg)

Modified Contamination Factor is a multielement index used to assess soil and sediment contamination. This classifies the soil on a seven-order scale of contamination, which is: very low contamination (mCdeg ≤ 1.5), Low contamination (1.5 ≤ mCdeg ≤ 2), Moderate contamination (2 ≤ mCdeg ≤ 4), High contamination (4 ≤ mCdeg ≤ 8), Very high contamination (8 ≤ mCdeg ≤ 16), Extremely high contamination (16 ≤ mCdeg ≤ 32), Ultra-high contamination (32 ≤ mCdeg) ( Source ).
